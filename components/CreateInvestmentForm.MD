```markdown
// Fields for the Investment Instrument Form:

**Organization Name**

- **Type**: Text input (string)

**Type of Investment Instrument**

- **Type**: Dropdown menu (select)
- **Options**: "Deposit", "Government Bond", "Corporate Bond"

**Related Comments**

- **Type**: Text input (string)
- **Example**: Instrument code or other relevant information

**Investment Currency**

- **Type**: Dropdown menu (select)
- **Options**: "USD", "EUR", "MDL", "Other"

**Investment Amount**

- **Type**: Numeric input (number)

**Rate of Return**

- **Type**: Numeric input (number)
- **Example**: 5% annual return

**Monthly Income**

- **Type**: Numeric input (number)
- **Example**: Expected monthly yield

**Instrument Maturity Date**

- **Type**: Date picker (date)
- **Example**: Calendar for specifying the expiration date

**Instrument Status**

- **Type**: Dropdown menu (select)
- **Options**: "Expires in 3 months", "Expires in 1 month", "Expired"
```
